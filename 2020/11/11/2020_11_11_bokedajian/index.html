发表于2020-11-11|更新于2020-11-11
|阅读量:

本博客就是基于 Hexo 在 GitHub上搭建的, 你不需要有任何的建站和代码知识, 只需要有一颗善于探索心和不轻易放弃的精神, 就能收获一个专属于你自己的互联网名片——博客, 下面就一起来学习一下, 如何操作吧!

前言

由于博客全部搭建在GitHub上, 所以访问会比较缓慢, 图片加载需要大家耐心等待!

  • 如果在搭建环境的过程中, 长时间没有反应，请尝试开启 科 学 上 网

注册 GitHub 账号

注册过程有手就行, 点击注册链接: https://github.com/join?source=header进入注册, 实在不会百度一下.

创建仓库

访问: 创建入口 github.com/new

创建一个仓库, 用于托管博客代码

  • 仓库的格式必须为：<用户名>.github.io
  • Description：为描述仓库（选填）
  • 勾选 Initialize this repository with a README 初始化一个 README.md 文件
  • 点击 Creat repository 进行创建

访问你的网站

打开浏览器，输入网站：用户名.github.io , 例如：PolarWBL.github.io

看到你的网站就意味着你成功建好仓库啦!
